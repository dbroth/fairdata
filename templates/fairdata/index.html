Your Data

Save your data as a .csv (Comma Separated Values) file. The first row of information should contain the names of data columns. After the first row, there should neither be any blank entries, nor entries after the last column. Please see this file for an example of the formatting our tool expects. The tool requires columns marked as Unprotected to contain only data that can be ordered and averaged; at this time, this means only numerical data.

Once you upload the file, mark the columns in the form according to the following definitions.

Identifier columns contain information that should be irrelevant to a classifier. These columns will be ignored.

Examples: applicant IDs, phone numbers.

Protected columns contain information that should not be used in a classifier.

Examples: race, gender, religion.

Unprotected columns contain information that should be used in a classifier.

Examples: interview scores, years of experience.

Class columns contain the outcomes assigned to applicants. There must be exactly one Class column per data set.

Examples: yes/no, good/bad.

Stratify columns contain values of groups which should be treated equally by a classifier. There must be at least one Stratified column per data set.

Finally, enter an email and press Calculate! Your email will only be used to send a URL to the repaired results. Your email will not be stored.
